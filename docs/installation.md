# Installation

## Stable release

To install Python Boilerplate, run this command in your terminal:

```sh
uv add openllm-prompt-mender
```

Or if you prefer to use `pip`:

```sh
pip install openllm-prompt-mender
```

## From source

The source files for Python Boilerplate can be downloaded from the [Github repo](https://github.com/diqiuzhuanzhuan/openllm-prompt-mender).

You can either clone the public repository:

```sh
git clone git://github.com/diqiuzhuanzhuan/openllm-prompt-mender
```

Or download the [tarball](https://github.com/diqiuzhuanzhuan/openllm-prompt-mender/tarball/master):

```sh
curl -OJL https://github.com/diqiuzhuanzhuan/openllm-prompt-mender/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd openllm-prompt-mender
uv pip install .
```
